## Συνδεσμολογία

MCU to GSM module
---------------------------
TX: 5 -> RX
RX: 15 -> TX
Power: 5 -> POWER

MCU to SDI12 Arduino Nano Slave
---------------------------------
SDA: 32 -> 27 (A4)
SCL: 33 -> 28 (A5)

MCU to Water Level sensor
----------------------------------
Analog: 36 -> AN
Power: 37 -> Solar Manager 3.3V on/off?

SDI12 Arduino Nano slave to Aquatroll
---------------------------------------
D5 -> Aquatroll SDI12
